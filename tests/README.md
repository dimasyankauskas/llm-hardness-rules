# Legacy Tests

This directory contains historical experiments and report artifacts from earlier iterations of the project.

Use `benchmarks/` for the current supported evaluation harness.

- `benchmarks/run_suite.py` is the active runner
- `benchmarks/suites/core_behavior.json` and `benchmarks/suites/interactive_tool_use.json` are the active suites
- timeouts in the new harness count as failures

Keep this directory for historical context, not for current public claims.
